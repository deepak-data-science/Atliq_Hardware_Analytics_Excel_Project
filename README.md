## 📌 Overview
AtliQ Hardware is a global hardware manufacturing company producing products such as PCs, printers, and accessories. It operates through both offline and online distribution channels.

This project focuses on building a **Business Intelligence solution using Excel** to analyze sales and financial performance and generate actionable insights.

---

## 🎯 Objective
To design an end-to-end analytics solution that:
- Tracks key business metrics  
- Enables performance analysis across markets and products  
- Supports data-driven decision-making  

---

## 🛠️ Tech Stack
- **Excel**
  - Power Query (ETL)
  - Power Pivot (Data Modeling)
  - DAX (Calculations)
  - Pivot Tables (Reporting)

## 🔄 Workflow
1. Data Collection from multiple sources  
2. Data Cleaning & Transformation using Power Query  
3. Data Modeling with relationships in Power Pivot  
4. KPI creation using DAX  
5. Dashboard & Report generation  

---

## 📊 Key Reports
- Customer Performance Report  
- Market Performance Report  
- Market vs Target Analysis  
- Top Products Analysis  
- Division-Level Performance  
- Profit & Loss (P&L) Reports  

---

## 📈 Sales Insights
- Top 10 products identified based on sales growth (2020–2021)  
- Strong growth across divisions:
  - PC: 313.7%  
  - P&A: 221.5%  
  - N&S: 84.4%  
- India emerged as the top-performing market  

---

## 💰 Financial Insights
- Consistent year-over-year growth in revenue and margins  
- Regional profitability trends identified using GM%  
- India leads in total revenue, while New Zealand shows highest margin %  

---

## 🚀 Key Learnings
- Built scalable data models using Power Pivot  
- Applied DAX for business KPIs  
- Designed interactive dashboards in Excel  
- Translated raw data into business insights  

---

## 📌 Conclusion
This project demonstrates how Excel can be leveraged as a powerful BI tool to analyze large datasets, generate insights, and support strategic business decisions.
